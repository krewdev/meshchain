# MeshChain press kit

## Boilerplate (short)

MeshChain is open-source, privacy-forward software that adds a mesh-native value layer to Meshtastic LoRa networks. Users can hold and move testnet balances off-grid, keep cold keys air-gapped, and bind Solana devnet vault deposits so unlock requires mesh-side proof—not the internet alone. Public testnet `meshchain-testnet-1` includes faucet, block scanner, and documentation. tMESH has no cash value. MeshChain is community software and is not an official Meshtastic product.

## Boilerplate (long)

MeshChain explores settlement when infrastructure fails or when users refuse always-on financial surveillance. Built for LoRa airtime limits, the stack includes a proof-of-authority style mesh ledger (tMESH on testnet), CLI wallets with memorable mesh names, multi-validator gossip for lab and field experiments, quantum-resistant cold signing (ML-DSA-65) for large moves, and a hybrid vault program on Solana devnet that binds deposits to mesh identity and requires multi-attestor co-sign for cash-out.

The project ships a public website, faucet, and blockchain scanner. All software is open source under the repository at github.com/krewdev/meshchain. The current network is experimental testnet only; tokens have no cash value and must not be marketed as investments.
