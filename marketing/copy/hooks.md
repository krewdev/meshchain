## Reply templates

**Curious builder**
> Happy to help. Fastest path: docs → testnet-setup → faucet → watch height on the scanner. Repo is Apache/MIT-style open source under krewdev/meshchain.

**Skeptical mesh op**
> Fair. Airtime budget is real. The ledger is designed for LoRa constraints; large PQ sigs use multi-packet FRAG. Critique welcome on GitHub Issues.

**Crypto-only audience**
> Think “settlement path that can survive internet failure,” not another L1 yield farm. Vault cash-out needs mesh burn + multi-attestor — internet alone fails by design.
